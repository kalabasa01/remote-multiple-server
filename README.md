# Remote Multiple Server via SSH

Simple Pyhon Script that will help to run command for multiple linux servers.
- Get Firewall Information
  - Get all the whitelisted Public IPs, Port and Protocol.
- Get Services Information
  - This will get the status of the services like Service Name, if active and On-Boot information.
  - This will also force the service to start and enable remotely.

## Language Used:
- Python

## Python Library:
- Netmiko
